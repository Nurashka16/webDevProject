# Лабораторные работы по JavaScript

Набор лабораторных работ по курсу JavaScript, включающий реализацию интерактивных веб-приложений с использованием современных возможностей языка и веб-API.

## Содержание

- **Лабораторная №1** — основы DOM и событий  
- **Лабораторная №2** — работа с кнопками, корзиной, сортировкой и координатами  
- **Лабораторная №3** — валидация формы с использованием `Promise` и `dialog`  
- **Лабораторная №4** — работа с промисами, генерацией данных и асинхронным крафтом  
- **Лабораторная №5** — `fetch`, `localStorage`, тосты, переключение темы  
- **Лабораторная №6** — классы, наследование, `EventSource`, паттерн "Наблюдатель"
